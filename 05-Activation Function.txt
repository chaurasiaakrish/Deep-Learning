Neurons take inputs, do some math, and then give an output.

There are 3 components that controls the process:


1:- ACTIVATION FUNCTION


What is an Activation Function?
An activation function decides whether a neuron should “activate” (fire) or stay silent after doing its weighted calculation.
It is basically Neuron's Decision Maker.

Without activation functions, a neural network becomes a long linear equation, even if it has many layers.
Meaning: it cannot learn complex or curved patterns.

Activation functions add non-linearity, allowing neural networks to learn things like:

curves
complex patterns
shapes
images
language structure

Example: A simple activation function is the "ReLU" (Rectified Linear Unit). If the neuron's calculation is positive, ReLU outputs that value directly.
If the calculation is negative, ReLU outputs zero. So, it only activates if the input is positive. 

- If input > 0 → output = input  
- If input ≤ 0 → output = 0  
ReLU is simplest activation function and it is simple, fast, and works very well in deep networks.
